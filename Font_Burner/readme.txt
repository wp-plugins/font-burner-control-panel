The Font Burner Plugin allows you to easily add Font Burner Fonts to your Wordpress theme.

== Description ==
The Font Burner Plugin allows you to easily add <a href="http://www.fontburner.com/the-font-burner-wordpress-plugin/">Font Burner</a> Fonts to your Wordpress theme.

== Installation ==

Installing Font Burner is very easy and do not require any template modification. Just follow these steps :

1. Upload the folder 'Font_Burner' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on "design" and you should see Font Burner Options at the top of the page. Click this and follow the directions to add the font of your choice to your site.

It's that simple!
